# Install

## Applications

The following applications will be installed:

- neofetch
- ranger
- git
- lsd
- bpytop
- htop
- zsh
- toilet
- figlet

## Repositories

The following repositories should be cloned (and installed as necessary):

- gh
- ohmyzsh

## Configs

The following configuration files should be copied to ~/:

- .zshrc
- .bashrc
- .config/neofetch
- .config/bpytop/bpytop.conf
